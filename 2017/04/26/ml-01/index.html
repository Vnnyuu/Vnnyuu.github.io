机器学习综述

监督学习

这类算法必须知道预测什么，即目标变量的分类信息。
我们给 学习算法 一个数据集（正样本），这个数据集由“正确答案”组成，机器可以从中推演出制定目标变量的可能结果。

监督学习 描述
分类问题 将实例数据划分到合适的分类中。例如：肿瘤检测问题。
回归问题 预测数值型数据。例如：房价问题。

监督学习算法

算法 描述（学到补充）
K-近邻
朴素贝叶斯
支持向量机
决策树
线性回归
局部加权线性回归
Ridge回归
Lasso最小回归系数估计

无监督学习

数据没有类别信息。也不会给定目标值。

无监督学习 描述
聚类问题 将数据集合分成由类似的对象组成的多个类的过程。
密度估计 寻找描述数据统计值的过程。
降维 减少数据特征的维度。

无监督学习算法

算法 描述（学到补充）
K-均值
最大期望值算法
DBSCAN
Parzen窗设计

算法如何选择

先看是否需要预测目标变量值

需要：监督学习

目标变量类型 算法类型
离散型 分类算法
连续型 回归算法

不需要：无监督学习

算法类型 应用
聚类算法 只需将数据划分为离散的组
密度估计算法 还需估计数据与每个分组的相似程度

解决问题的步骤

步骤 描述
1.收集数据 收集数据样本。
2.处理数据 确保数据格式符合要求，为算法准备特定的数据格式。
3.分析输入数据 确保数据集中没有垃圾数据。
4.训练算法 跑算法，将前三步得到的数据输入到算法中，从中抽取知识或信息。无监督学习没有前三步。
5.测试算法 实际使用前四步得到的知识信息。测试算法的工作效果。如果效果不尽人意，需要调到前几步进行。
6.使用算法 将算法实用化。
